## 🔍 DETALHES TECNICAS DO SELETOR

### Problema Original
```
Google Maps atualizou sua estrutura HTML em 2025
Seletor antigo: div[data-item-id] (descontinuado)
Resultado: 0 elementos encontrados
```

### Solucao Implementada
```javascript
// Seletor novo (testado e validado)
const buttons = document.querySelectorAll('button[aria-label]');

buttons.forEach(button => {
    const label = button.getAttribute('aria-label');
    
    // Filtro preciso: comeca com "Ver rotas para"
    if (label.startsWith('Ver rotas para')) {
        // Extrair nome e processar
        let nome = label.replace(/^Ver rotas para\s*/, '').trim();
        // ... continua processamento
    }
});

// Resultado: 30+ empresas por sessao
```

### Como Funciona
1. Playwright carrega Google Maps com stealth mode
2. Aguarda 3 segundos para carregamento completo
3. Executa JavaScript para encontrar todos os buttons
4. Filtra buttons que sao resultados de busca
5. Extrai nomes, ratings, enderecos
6. Retorna lista de EmpresaBase

## 📞 SUPORTE E TROUBLESHOOTING

### Problema: "Executavel nao funciona"
**Solucao:** Verifique se Windows Defender/Antivirus esta bloqueando. Adicione.exe a whitelist.

### Problema: "Rate limit na API ReceitaWS"
**Solucao:** Normal - API tem limite de 20 requisicoes/minuto. Sistema implementa exponential backoff automaticamente.

### Problema: "Google Maps nao carrega"
**Solucao:** Verifique conexao internet. Se problema persistir, execute:
```cmd
python debug_maps_selectors.py
# Isso vai validar conectividade e seletores
```

### Problema: "CSV vazio"
**Solucao:** Verifique se Google Maps retornou resultados. Tente busca diferente ou mais genérica.
